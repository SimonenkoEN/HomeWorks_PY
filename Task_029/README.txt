Разработка Каринэ Симоненко в соавторстве с Евгением Симоненко
При написании программы за структурную основу была взята программа, написанная на семинаре. Однако, код написан с нуля.
Функции программы в требуемых местах содержат минимальные проверки вводимых данных.
interface - начинается с блока проверки существования файла БД. После выполнения выбранного действия автоматический возврат в главное меню.
add_new_line - id задается автоматически, что исключает повторы. Есть проверка на введение повторяющихся записей. После введения выводится запрос на сохранение или ввод заново.
print_to_console - вывод организован в виде форматированной таблицы. Организована сортировка вывода по содержанию. Если БД пустая выводится сообщение об отсутствии данных для вывода.
search - в функции поиска пользователю предлагается выбрать раздел базы данных для поиска, а далее выводится запрос на искомую запись. При нулевом результате поиска выводится соответствующее сообщение.
edit_line - при редактировании функция дает возможность пользователю найти необходимую запись, и затем, используя id нужной записи, проходит по всей записи, предлагая заменить необходимый раздел.
delite - при удалении, пользователю дается возможность найти необходимую запись, и далее введением ее id осуществляется подтверждение действия.
